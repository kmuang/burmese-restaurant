# burmese-restaurant

This repository contains a React app for a Burmese restaurant project.

## Getting Started

In the project directory, install dependencies and run the app:

### `npm install --legacy-peer-deps`

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Notes

- The app was created with Create React App.
- Local changes have been merged with the existing GitHub repository content.
